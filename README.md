# 📚 Leviora
*Read without distractions. Focus without limits.*

---

## 🌱 About

Leviora is a next-generation offline PDF reader for Android designed to deliver a smooth, immersive, and distraction-free reading experience.

It goes beyond a simple document viewer by combining powerful reading tools, elegant glass-inspired UI, and high-performance rendering — all while keeping everything fully offline and private.

## ✨ Features

### 📚 Advanced Library Management
- Combine multiple PDFs into a single **Book**
- Custom book covers from gallery
- Organize using categories, tags, and sorting
- Track reading progress and completion
- Edit or replace PDFs without losing progress

---

### 📖 Immersive Reading Experience

#### Reading Modes
- Vertical Scroll (smooth, continuous)
- Horizontal Swipe (page-by-page)
- Flipbook Mode (realistic page animation)

#### Core Features
- Instant resume (page, zoom, position)
- Fast page navigation slider
- Tap-to-hide UI (distraction-free mode)
- Table of Contents support

---

### 📝 Annotation System
- Add notes to specific pages
- Sticky notes with precise placement (X/Y position)
- Notes dashboard for quick access
- Fast navigation to saved notes

---

### ⏳ Focus Mode
- Built-in reading timer
- Custom time selection (scroll wheel picker)
- Minimal UI for deep focus
- Session summary with time and pages tracked

---

### 👻 Ghost Mode (External Reader)
- Open PDFs directly from file manager, browser, or other apps
- No need to import into library
- Temporary session (no data saved)
- Full feature access during session

---

### 📊 Smart Tracking & Insights
- Track reading time per book
- Monitor pages read and completion percentage
- Weekly reading insights
- All data stored locally (no tracking)

---

### 🎨 Premium Glassmorphism UI
- Custom glass-style components (GlassCard, GlassSurface)
- Layered UI with depth and smooth visuals
- Clean, minimal, distraction-free design

#### Themes:
- Dark Mode (default)
- OLED Pure Dark
- Light Mode

---

### ⚙️ Settings & Personalization
- Theme switching (Light / Dark / OLED)
- Toggle reading insights visibility
- Keep screen on while reading
- Tap-to-turn control
- Persistent user preferences

---

## ⚡ Performance & Optimization

Leviora is engineered for performance across all devices:

- Smooth PDF rendering engine
- Lag-free scrolling even for large PDFs (2000+ pages)
- Memory-efficient bitmap handling (RGB_565)
- Adaptive caching based on device memory
- Stable performance on low-end devices (tested on 1GB RAM)
- Long reading session stability (tested 4+ hours)

## 🔒 Offline & Privacy First

- Fully offline (no internet required)
- No data collection
- No cloud sync
- All data stays on your device
